ClassicalParticles

LennardJonesEpsilon
Section: ClassicalParticles
Type: float

Epsilon parameter (dispersion energy) of Lennard-Jones interaction for this species. In case two particles have a different epsilon, the combination rule will be computed \(\epsilon_{12} = \sqrt{\epsilon_1 + \epsilon_2}\).

LennardJonesSigma
Section: ClassicalParticles
Type: float

Sigma parameter (particle size) of Lennard-Jones interaction for this species. In case two particles have a different sigma, the combination rule will be computed \(\sigma_{12} = (\sigma_1 + \sigma_2) / 2.
